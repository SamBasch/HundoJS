                    function fetchUserInput() {
                        //get values from page
                    
                    
                        let startValue = document.getElementById('startValue').value;
                    
                        let endValue = document.getElementById('endValue').value;
                    
                    
                    
                    
                        //parse our inputs as numbers
                    
                        startValue = parseInt(startValue);
                        endValue = parseInt(endValue);
                    
                    
                    
                        //verify our inputs are actually numbers (validation)
                    
                        if (Number.isInteger(startValue) && Number.isInteger(endValue)) {
                    
                                let numbersArray = generateNumbers(startValue, endValue);
                    
                                displayNumbers(numbersArray);
                    
                    
                        } else {
                    
                            Swal.fire(
                                {
                                    icon: 'error',
                                    title: 'Oops!',
                                    text: 'Only integers are allowed for Range Finder!'

                            );
                    
                    
                        }

                    }
                    
                        
                    
                    
                    
                    //generate our numbers
                    
                    function generateNumbers(start, end) {
                    
                        let numbers = [];
                    
                        for(let i = start; i <= end; i++) {
                    
                            numbers.push(i);
                        }
                    
                        return numbers;
                    
                    
                    
                    }
                    
                    
                    
                    //display them on page
                    
                    //view funciton (function is dedicated to making it 'show up' on our web page)
                    function displayNumbers(numbersArray) {
                            let tableBody = document.getElementById('results');
                    
                    
                    
                            let tableHtml = "";
                    
                            for(let i = 0; i < numbersArray.length; i++) {
                                
                                let value = numbersArray[i];
                                let className = '';
                    
                                if (value % 2 == 0) {
                                    className = 'even'
                                } else {
                                    className = 'odd'
                                }
                    
                    
                                if(i % 5 == 0) {
                                    tableHtml = tableHtml + ''
                                }
                    
                                let tableRow = `${value}`
                    
                                tableHtml = tableHtml + tableRow;
                    
                                if (i + 1 % 5 == 0) {
                                    tableHtml = tableHtml + ''
                                }
                                
                            }
                            
                            tableBody.innerHTML = tableHtml;
                    
                        }

generateNumbers()

This is the second function. This function is responsible for looping through all the of the numebers inbetween the start and end number created by the user via a for loop (the for loop also includes the start and end numbers themselves). As the for loop increments throughout each number in between the start and end value it appends each value on to the end of the empty array we created called 'numbers'. Once the for loop has pushed all values on to the empty array it returns that array making the array equal to the variable we put to the left of the function call in the first function. displayNumbers()

This function is the third function. This function is called (by the first function) when the first function receives the array from the second function. This function is responsible for utilizing a for loop to display the numbers in form of a table (which rows are customizable via modulo) on the web page and looping through all the numbers to assign even and odd numbers their respective class we made in CSS so they display differently.

First, we declare the variable 'tableBody' and set it equal to the element we gave the id "results" This is a space we designated on the html page to house our table and the content of it

then, we declare the variable 'tableHtml' as an empty string. This empty string variable is going to act as an 'accumlator' for all of our table rows and data and numbers within them.

Within the for loop itself we declared a variable named 'value' this variable represents the current number that the forloop is on. I also declared another variable called 'className' as an empty string. This empty string will get assigned 'even' or 'odd' determined by an if statement checking to see if that number is divisble by 2 via %.

The for loop then checks to see if the current index of the for loop is divisble by 5 or not via %. If the current index is divisable by 5 then it will afix a table row opening tag to the beginning of the data. This is how I customized how many data entries there were per row.

We then used a string template literal which is assigned to a variable called 'tableRow'. With every loop, The tableRow variable containing the string template literal (which contains the current number the for loop is on) is then added to the tableHtml empty string variable, thereby creating all the entries to our table with every loop

Another if statment check to see if the NEXT index or the index AFTER the one we are on is divisble by 5, if it is then a closing table row tag is appended to the previous table data entry as to close one row and form the other. If you changed the number after the modulos you can customize how many data entries there are per row.

For the final touch we declare that the 'innerHTML' of the tableBody is equal to the variable tableHtml. This will display all the table rows and table data created by the for loop in the space we designated on the html page
